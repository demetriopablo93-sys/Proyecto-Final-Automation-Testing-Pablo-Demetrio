# Proyecto Integrador de Automatización - Talento Tech - Pablo Demetrio(ENTREGA FINAL)

Automatización de UI + API utilizando __Python__, __Selenium__, __Pytest__ y __GitHub Actions__.

## Propósito del proyecto
Desarrollar un framework de pruebas automatizadas en Python que cubra UI (Selenium) y API (Requests), con estructura modular, generación de reportes HTML y captura de pantallas en fallas.


## Tecnologías
- Python
- Pytest
- Selenium WebDriver
- Requests
- pytest-html
- webdriver-manager
